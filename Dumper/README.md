# Offset dumper (optional)

DMA-based helper for finding GTA5 Enhanced offsets after a game update.

1. Build/run this dumper on the DMA PC with the game running on the game PC  
2. Copy the new values into `GTA5_DMA/Offsets.txt`  
3. Restart `GTA5_DMA.exe` — no rebuild needed for offset-only changes  

Symbol cache under `Symbols/` is gitignored (large / machine-specific).
